Self hosting of Mee6 the bot (dev version)

Host pc

Hosting Mee6 on a Windows maching is not easy, so i suggest using a linux machine.
For this tutorial i made use of a ubuntu VM in Virtualbox

preperation

database

Be sure your linux machine is up to date.
Download and unpack redis db.
Type make in the redis directory.
now you should have a workin redis server, start it with:
./src/redis-server ./redis.conf
Use Ctrl + C to quit the redis server.

site

Clone the git repo
git clone https://github.com/cookkkie/mee6 && cd mee6
Create a application on Discord
Add these 2 redirect uri's:
http://localhost:5000/confirm_login
http://localhost:5000/servers
Run the following commands (use pip3 if pip is bound to your python2 instance)
cd website && pip3 install -r requirements.txt

bot

Run the following commands (use pip3 if pip is bound to your python2 instance)
cd chat-bot && pip install -r requirements.txt
python3 -m pip install -U https://github.com/rapptz/discord.py/archive/async.zip#egg=discord.py[voice]
Create a bot user in your app.

variables

Add this to your .bashrc file (on the bottem)
(this file can be found in the user directory)


export REDIS_URL=redis://localhost  

export OAUTH2_CLIENT_ID=123  

export OAUTH2_CLIENT_SECRET=123  

export MEE6_TOKEN=123  

export OAUTH2_REDIRECT_URI=http://localhost:5000/confirm_login  

export MAL_USERNAME=123  

export MAL_PASSWORD=123  

export TWITCH_CLIENT_ID=123  

export GOOGLE_API_KEY=123  

# Flask secret key  

export SECRET_KEY=1234

Fill in the needed values.
Leave the first one.
For the next 4 you need to get them from your Discord app
The flask secret key is just a random value (change it!!! security!!)
For the others... idk, i make them empty.

running the bot

First RESTART YOUR SHELL.
start your redis server as mentioned in the redis install part Run the website: cd mee6/website && python3 app.py Run the bot cd mee6/chat-bot && python3 bot.py

Note

the paypalrestsdk does not seems to install always, you might want to put line 10 in mee6/website/app.py in commentary (put in # in front)

you can now go to your personal dashboard (on the pc you are hosting the bot on)
